XJX Library Examples (ES Modules)

Instructions

This page demonstrates the XJX library for XML-JSON transformation using various transformer types. Open your browser's developer console to see the output.

Press F12 or Ctrl+Shift+J (Windows/Linux) or Cmd+Option+J (Mac) to open the console.

Note: ES Modules only work when served from a proper web server, not directly from the file system.

Transformer Types

Value Transformers

Modify primitive values like strings, numbers, and booleans. Examples include converting string "true" to boolean true, or string "123" to number 123.

Attribute Transformers

Modify attribute names and values. Examples include standardizing units (cm â†’ centimeters) or formatting currencies.

Node Transformers

Modify entire elements, including renaming them or adding attributes. Examples include renaming "title" to "name" or adding currency attributes to prices.

Children Transformers

Process collections of child nodes, allowing filtering, sorting, or reorganizing. Examples include sorting items by ID or filtering out discontinued products.
